# prestige590

A luxury single-page sales website for the **2023 Prestige 590 Flybridge** yacht ($1,790,000) in Marina Del Rey, CA.

Built with [Next.js](https://nextjs.org), TypeScript, Tailwind CSS v4, Framer Motion, and deployed on [Vercel](https://vercel.com).

## Getting Started

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

Set `NEXT_PUBLIC_SITE_URL` to your production domain so canonical tags,
Open Graph URLs, `robots.txt`, and `sitemap.xml` all resolve correctly after
deployment.

## Deploy

Click **Deploy** in Vercel or push to the linked GitHub repository — Vercel will auto-detect the Next.js project and deploy it.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4 + custom dark navy / gold theme
- **Animations**: Framer Motion (subtle scroll-triggered)
- **Icons**: Lucide React
- **Fonts**: Playfair Display + Inter (Google Fonts)
